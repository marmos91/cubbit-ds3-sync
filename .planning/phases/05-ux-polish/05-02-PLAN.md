    - Test 3: SharedData.loadPauseState returns empty dict when no file exists
    - Test 4: SharedData.setDrivePaused(driveId, true) then isDrivePaused(driveId) returns true

    **DS3DriveStatus (DS3Drive.swift):**
    - Add `case paused` to DS3DriveStatus enum (after `error`)
    - This is backward-compatible since it's Codable with String raw value

    **AppStatus.swift:**
    - Add `case paused` to AppStatus enum
    - Add paused case to toString(): return NSLocalizedString("Paused", comment: "Paused status")

    **SharedData+pauseState.swift:**
    - Create new file extending SharedData
    - Persistence file: `pauseState.json` in App Group container (same pattern as other SharedData extensions)
    - `public func setDrivePaused(_ driveId: UUID, paused: Bool) throws` -- loads current state dict, updates entry, saves. Uses NSFileCoordinator for cross-process safety (same pattern as SharedData+tenant.swift).
    - `public func isDrivePaused(_ driveId: UUID) throws -> Bool` -- loads state dict, returns value for driveId (default false). Uses NSFileCoordinator for reading.
    - `public func loadPauseState() throws -> [UUID: Bool]` -- loads full pause state dict
    - Internal storage: JSON-encoded `[String: Bool]` where keys are UUID strings
    - Handle missing file gracefully (return empty dict)

    **FileProviderExtension.swift:**
    - In the main CRUD methods (fetchContents, createItem, modifyItem) add a pause check AFTER the guard-let chain:
      ```swift
      if let drive = drive, (try? SharedData.default().isDrivePaused(drive.id)) == true {
          // Paused: reject new operations with a temporary error so the system retries later
          throw NSFileProviderError(.serverUnreachable)
      }
      ```
    - Use .serverUnreachable so the system will retry automatically when resumed
    - Do NOT add pause check to deleteItem (deletes should complete even when paused)
    - Do NOT add pause check to enumerateItems (enumeration should continue to show file list)
    - The pause check is a lightweight SharedData read -- acceptable for each operation
    - Log when a paused drive rejects an operation: logger.info("Drive paused, deferring operation")

    **PauseStateTests.swift:**
    - Write tests per the behavior spec above
    - Use a temporary directory for SharedData to avoid polluting the real App Group container
    - Test Codable round-trip for DS3DriveStatus.paused
